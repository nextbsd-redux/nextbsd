# One pkg per line; lines starting with # are ignored.
# Curated FreeBSD-* pkgbase packages — runtime portion. Replaces the
# wholesale base.txz + kernel.txz extraction with an install of only what
# this ISO actually needs at boot/runtime. Installed host-side via
#   pkg -R $WORK/pkg-config -r $WORK/rootfs install -y ...
# against pkg.freebsd.org/FreeBSD:${FREEBSD_VERSION}:${ARCH}/base_latest.
#
# Build-only base pkgs (clang, lld, *-dev) live in buildpkgs-base.txt and
# are purged before mkuzip, same pattern as buildpkgs.txt today.
#
# Explicitly omitted (vs. base.txz contents): kerberos, lldb, dtrace,
# audit, bluetooth, hyperv-tools, mlx-tools, cxgbe-tools, nuageinit,
# bsdinstall, bsdconfig, yp, ppp, ftp, ipf, ipfw, pf, blocklist, lib9p,
# quotacheck, hast, ggate, natd, bsnmp, autofs, at, apm, acct, ee, dma,
# inetd, hostapd, kernel-man, mandoc, periodic, local-unbound, netmap,
# newsyslog, nfs, nvme-tools, rdma, rcmds, sound, tcpd, zfs, locales,
# libldns, libdwarf, libevent1, librpcsec_gss, libipt, libcuse, libvgl,
# libvmmapi, libbegemot, libbsdstat, libpathconv, librss, fwget, ctf,
# ccdconfig, freebsd-cert, efi-tools — these are the ISO-size savings.
#
# Removed 2026-05-16 (post Phase I3 PID-1 launchd boot to login):
# rc (launchd replaces /etc/rc), devd + devmatch (hotplug only, kernel
# does the initial probe; notifyd will replace eventually), ssh
# (defer to Apple openssh via launchd plist later), resolvconf
# (configd will handle DNS resolver state), syslogd (ASL will replace),
# libthread_db (gdb-only), libmagic (file(1)-only).

#   our ISO. Plan: https://pkgdemon.github.io/freebsd-srclist-build-plan.html §2
#   Historical: #104 (closed not-planned), PR #107 (closed unmerged).
